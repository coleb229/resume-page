CSV Comparison Tool

Link to Repo on Github

This is another tool that I created to help me at work. This is a simple yet powerful tool that allows me to compare two CSV files and see the differences between them. I created this tool because I often have to compare CSV files to find differences between them, and I wanted to create a tool that would make this process easier.

I used The Pandas library with Python to program the logic. On top of that, I used tkinter to display the output in a GUI window.
